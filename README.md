# 🎰 DotaRoulette

> Крути рулетку — выбирай героев для своей пати в Dota 2!

Веб-приложение для друзей, играющих в Dota 2, которое помогает случайным образом распределить роли и героев с помощью красивой анимации рулетки.

## ✨ Возможности

- **Авторизация** — демо-вход через Steam / Discord / Telegram / Dotabuff
- **Настройка пати** — выбор от 1 до 5 игроков, указание никнеймов
- **Выбор ролей** — ручной (Carry, Mid, Offlane, Support, Full Support) или автоматический (рандом)
- **3 режима подбора героев**:
  - 🛡️ **Сильная линия** — герои с мощной синергией на лейне
  - 🎉 **Весёлые комбо** — забавные и эффектные связки героев
  - 🏆 **На победу** — метовые герои для максимального импакта
- **Рулетка с анимацией** — слот-машина прокручивает героев, постепенно замедлясь
- **Звуковые эффекты** — тик-тик-тик при прокрутке + звук выбора
- **Перекрутка** — реролл одного героя или всех сразу
- **История** — сохранение всех прошлых роллов с датами
- **Страница 404** — тематическая страница с юмором Dota 2
- **Адаптивный дизайн** — работает на десктопе и мобильных

## 🛠️ Технологии

- **Frontend**: Vite + Vanilla JS
- **Backend**: Express.js
- **Database**: SQLite (better-sqlite3)
- **Auth**: JWT (httpOnly cookies)
- **Стиль**: CSS с glassmorphism, glow-эффектами, анимациями
- **Звук**: Web Audio API

## 🚀 Запуск

### Локально

```bash
# Установить зависимости
npm install

# Запустить в dev-режиме (сервер + клиент)
npm run dev
```

Откройте http://localhost:5173

### Продакшн

```bash
# Собрать фронтенд
npm run build

# Запустить сервер
npm start
```

## 📦 Деплой на Render

1. Пушим в GitHub: `git push origin main`
2. На [render.com](https://render.com) создаём Web Service
3. Подключаем репозиторий
4. Render автоматически использует `render.yaml` для настройки

**Или** используем Blueprint:
- Build Command: `npm install && npm run build`
- Start Command: `npm start`

## 🎮 Как пользоваться

1. Заходим на сайт, вводим никнейм
2. Выбираем количество игроков (1-5)
3. Указываем никнеймы и роли (или оставляем "Авто")
4. Выбираем режим подбора героев
5. Жмём "Крутить рулетку"!
6. Смотрим анимацию для каждого игрока
7. Видим результаты, можем перекрутить или сохранить
